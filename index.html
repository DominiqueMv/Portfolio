Conoce acerca de las tecnologías y herrramientas que utilizo.

Bootstrap

Imagina Bootstrap como un conjunto de piezas de LEGO especiales para construir sitios web. Estas piezas ya tienen formas y colores geniales, por lo que no tienes que empezar desde cero. Simplemente eliges las piezas que necesitas y las unes para armar tu sitio. Además, Bootstrap también te da una especie de guía para colocar las piezas de manera ordenada, así todo se ve bien en diferentes tamaños de pantalla.

HTML

HTML es como la estructura de un edificio para sitios web. Imagina que estás construyendo una casa con bloques de construcción. Cada bloque tiene un propósito, como un bloque para el título, otro para el texto y otro para las imágenes. Juntas, estas piezas forman la página web. También puedes crear caminos entre las casas usando bloques especiales para enlaces.

CSS

CSS es como la pintura y la decoración para tu sitio web. Después de construir la estructura con HTML, es hora de darle estilo con CSS. Imagina que tienes una casa (que sería tu página web) y quieres hacerla lucir genial. CSS te permite elegir colores, fuentes y diseños para que todo se vea bonito. Puedes cambiar el tamaño y la forma de las cosas, como si estuvieras redecorando una habitación.

GIT

Git es como una máquina del tiempo para tu código. Imagina que estás trabajando en un proyecto con un equipo y todos hacen cambios al mismo tiempo. Git te permite guardar versiones de tu trabajo en diferentes momentos, como tomar fotos del progreso. Si algo sale mal o quieres volver atrás, puedes retroceder en el tiempo a una versión anterior. También puedes trabajar en diferentes partes del proyecto sin interferir con los demás, como si cada persona tuviera su propia copia mágica.

GITHUB

GitHub es como una comunidad en línea para compartir y colaborar en proyectos de código. Imagina que estás trabajando en una historia con amigos, pero en lugar de pasar papel, usas una plataforma en línea donde todos pueden ver y editar la historia juntos. En GitHub, puedes subir tu código (como piezas de un rompecabezas) para que otros lo vean y contribuyan. Pueden hacer sugerencias o cambios, como añadir sus propias piezas al rompecabezas. También puedes llevar un registro de quién hizo qué y cuándo, como un diario de colaboración.
Figma
Figma es como una pizarra digital para diseñar cosas geniales. Imagina que estás creando un dibujo con amigos, pero en lugar de lápices y papel, todos usan una pizarra mágica en línea. En Figma, puedes dibujar botones, imágenes y todo lo que necesitas para un sitio web o una aplicación. Puedes trabajar con otros al mismo tiempo, como si todos estuvieran dibujando juntos en la misma pizarra. También puedes hacer maquetas interactivas, como un prototipo de cómo funcionará tu diseño.

Java

Java es como un motor detrás de escena que hace que las cosas funcionen en un sitio web. Imagina que estás organizando un festival y necesitas personas para llevar a cabo diferentes tareas: vendedores, músicos, seguridad, etc. En este caso, Java sería como el equipo de organización que coordina todas estas actividades. En el desarrollo web, Java se utiliza para crear aplicaciones web más complejas, como plataformas de comercio electrónico o sistemas de gestión. Ayuda a procesar información en el servidor y a interactuar con bases de datos, lo que permite a los usuarios ver y manipular datos en el navegador.
